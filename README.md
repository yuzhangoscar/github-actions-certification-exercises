# GitHub Actions Certification Exercises

This project is designed to provide quizzes and answers for readers who want to pass the GitHub Actions certification provided by Microsoft. It serves as a study guide and practice repository.

## NPM Modules Used

The following NPM modules are used to maintain project standards and code quality:

| Module | Version | Description |
| :--- | :--- | :--- |
| **husky** | [![npm version](https://img.shields.io/npm/v/husky.svg?style=flat-square)](https://www.npmjs.com/package/husky) | Modern native git hooks made easy. Used to manage Git hooks across the team. |
| **@commitlint/cli** | [![npm version](https://img.shields.io/npm/v/@commitlint/cli.svg?style=flat-square)](https://www.npmjs.com/package/@commitlint/cli) | Lint commit messages. Ensures commit messages meet the conventional commit format. |
| **@commitlint/config-conventional** | [![npm version](https://img.shields.io/npm/v/@commitlint/config-conventional.svg?style=flat-square)](https://www.npmjs.com/package/@commitlint/config-conventional) | Shareable commitlint config enforcing conventional commits. |
| **validate-branch-name** | [![npm version](https://img.shields.io/npm/v/validate-branch-name.svg?style=flat-square)](https://www.npmjs.com/package/validate-branch-name) | Validates git branch name. Enforces branch naming conventions (e.g., `feature/`, `fix/`). |
